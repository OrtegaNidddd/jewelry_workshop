# Proyecto de Taller de Joyería (Jewelry Workshop)

## Resumen del proyecto

Plataforma web para gestionar de forma integral los pedidos en talleres de joyería artesanal. Digitaliza el flujo desde la solicitud inicial hasta la entrega, centralizando cotizaciones, diseños y fabricación para asegurar trazabilidad y comunicación entre cliente, asesor y maestro joyero. Inicia como un MVP enfocado en las funciones esenciales.
